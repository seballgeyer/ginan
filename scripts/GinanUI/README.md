# Ginan-UI

An intelligent and user-friendly interface for using the Geoscience Australia GNSS processing tool Ginan. Made using PySide6 by students of the 2025 ANU TechLauncher program.

[User manual available here](https://drive.google.com/file/d/1ThoVqijgRQ1KV9QfP7m5iflhWILdRoD3/view?usp=drive_link)

## Installation
### From Source
Tested with python 3.9+
```
Navigate to the root of the Ginan repository:
cd /path/to/repo
pip install -r scripts/GinanUI/requirements.txt
python -m scripts.GinanUI.main
```
